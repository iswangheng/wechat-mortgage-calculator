# WeChat MiniProgram Project 01

## 项目概述
微信小程序 - 工具类应用，用于承接微信搜索流量并通过广告变现。

## 当前状态
🚀 **开发中** - 房贷计算器 2026

## 项目定位
**房贷计算器 2026** - 抓住2026年房地产政策红利期，为购房者提供专业的房贷计算工具

## 核心功能
- 商业贷款/公积金/组合贷款计算
- 等额本息 vs 等额本金对比
- 提前还款计算器
- 2026最新LPR利率
- 购房税费计算
